| Use Case               | Description                               |
| ---------------------- | ----------------------------------------- |
| **Authenticate User**  | Customer inserts card and enters PIN      |
| **Withdraw Cash**      | Dispense requested cash if balance allows |
| **Deposit Cash**       | Accept cash/check and update balance      |
| **Balance Inquiry**    | Display current account balance           |
| **Transfer Funds**     | Move money between accounts               |
| **Change PIN**         | Update customer PIN                       |
| **Mini Statement**     | Print recent transactions                 |
| **Cancel Transaction** | Stop current operation                    |
| **Eject Card**         | Return ATM card to customer               |

Use Case Relationships
Withdraw Cash → includes → Authenticate User
Deposit Cash → includes → Authenticate User
Transfer Funds → includes → Authenticate User
Change PIN → extends → Authenticate User

Customer
   |
   |-- Authenticate User
   |-- Withdraw Cash
   |-- Deposit Cash
   |-- Balance Inquiry
   |-- Transfer Funds
   |-- Change PIN
